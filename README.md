# Travel World 🌍✈️

A full-stack travel booking web application developed using PHP, MySQL, HTML, CSS, and JavaScript. Travel World allows users to explore destinations, book travel packages, reserve hotels, make payments, and generate invoices through a modern and responsive interface.                                                                                                                                                                            Website URL - https://travel-world.10001mb.com/

## 🚀 Features

### 👤 User Authentication
- User Registration
- User Login
- Secure Password Hashing
- Logout Functionality
- Session Management

### 🌍 Travel Package Booking
- Explore Popular Destinations
- View Package Prices
- Select Package Type (Silver, Gold, Premium)
- Choose Travel Date and Number of Travelers
- Booking Confirmation

### 🏨 Hotel Booking
- Hotel Selection
- Check-in & Check-out Dates
- Room and Guest Selection
- Hotel Booking Confirmation
- Hotel Invoice Generation

### 💳 Payment System
FAKE 
- UPI Payments
- Debit Card Payments
- QR Code Payments
- Payment Records Storage
- Booking Invoice Generation

### 📄 Invoice Management
- Travel Booking Invoice
- Hotel Booking Invoice
- Printable Invoice Pages

### 📬 Contact System
- Contact Form
- User Message Storage

### 📱 Responsive Design
- Mobile Friendly UI
- Modern Dashboard
- Interactive Booking Modals

## 🛠️ Technologies Used

### Frontend

### Backend
- PHP

### Database
- MySQL

## ⚙️ Installation

1. Clone the repository.
2. Place the project inside WAMPP www.
3. Create a MySQL database named `travel_world`.
4. Update database credentials in `db.php`.
5. Start Apache and MySQL.
6. Open the project in your browser.
